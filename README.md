### 🌍 Was ich mache

Seit mehreren Jahren arbeite ich an der Schnittstelle von **Hydrogeologie, Geomatik und Datenanalyse**: Grundwassermodellierung (FEFLOW, PEST), räumliche Datenanalyse, Geodatenbank-Design und die Automatisierung von Auswerte- und Reporting-Workflows mit Python und R. Mein Ansatz ist prozessorientiert — erst das System verstehen, dann modellieren, dann die Ergebnisse fachlich kritisch validieren.

Seit etwa einem Jahr liegt mein Fokus zunehmend auf **angewandter KI**: Entwicklung eigener Skills und Agenten, Testen von Pipelines für GIS-Endanwender und das Orchestrieren von Prozessen mit KI-Werkzeugen. Mein Mehrwert: **Domänenwissen + KI-Kompetenz + kritische Bewertung** — ich kann beurteilen, ob ein Ergebnis fachlich (hydrologisch, räumlich) korrekt ist, nicht nur, ob der Code läuft.

### 🔧 Womit ich arbeite

- **Modellierung &amp; Geodaten:** FEFLOW, PEST, QGIS, Smallworld GIS, räumliche Analysen, DEM-Verarbeitung
- **Programmierung:** Python (Datenverarbeitung, Automatisierung), R &amp; R Shiny (Statistik, Zeitreihen, interaktive Apps), SQL
- **Workflow &amp; KI:** Git/GitHub, automatisierte Pipelines, Prompt Engineering, Skill- &amp; Agentenentwicklung
